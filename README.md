# neon-dash-architect
Paper - 01 "Guía práctica para usar metodología Skills en el desarrollo aplicaciones y juegos"
Autor: Ing. Ronny Díaz López
Quito, Ecuador
Martes, 23 de Junio de 2026

---
Esta guía transforma la metodología de "Skills" (sin necesidad de Multiagentes) de Anthropic usada en Claude.ai para ser aplicada utilizando el framework Free OpenCode (u otros entornos de código abierto), utilizando modelos de IA gratuitos (como Minimax, Big Pickel, Ollama, DeepSeek V4 Flash Free, MiMo v2.5, North Mini Code Free, Nemotron 3 Ultra, y más) para desarrollar y expandir por ejemplo este caso de uso: “Juego Neon Dash”.

Opencode es un framework gratuito (con planes de pagos también) para el desarrollo de proyectos, opensouce, con opciones de desarrollo usando modelo tanto gratuitos como de pago. Estos modelos ya han sido “curados” probados por Opencode así que es un plus en el desarrollo de aplicaciones y demás proyectos. O mejor aún cómo ellos mismos lo dicen en su web: “Un agente de código abierto que te ayuda a escribir código en tu terminal, IDE o escritorio.” (https://opencode.ai/)
